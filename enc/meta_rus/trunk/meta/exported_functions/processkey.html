ProcessKey

главная | экспортируемые функции
Функция ProcessKey позволяет перекрывать стандартную обработку клавиш на панели плагина.
int WINAPI ProcessKey(
  HANDLE hPlugin,
  int Key,
  unsigned int ControlState
);

Параметры

hPlugin
Описатель плагина, возвращается функциями OpenPlugin или OpenFilePlugin.
Key
Виртуальный код клавиши.
ControlState
Состояние Shift-клавиш. Комбинация флагов (перечисление FAR_PKF_FLAGS):
ФлагОписание
PKF_CONTROL Клавиша Ctrl нажата
PKF_ALT Клавиша Alt нажата
PKF_SHIFT Клавиша Shift нажата
Например, при нажатии Shift-F7, Key= VK_F7 и ControlState = PKF_SHIFT.

Возвращаемое значение

FALSE для дальнейшей обработки FAR.
TRUE, если плагин самостоятельно обработал клавишу.

Замечания

Из-за особенностей реализации ядра FAR в эту функции приходят не все клавиши, нажимаемые пользователем. (верно для FAR до версии 1.70 beta 5 включительно).
