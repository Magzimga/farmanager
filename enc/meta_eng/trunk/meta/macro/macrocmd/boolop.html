ATTENTION!

  • Элементы макроязыка невозможно использовать во время записи макроса стандартным образом. Добавить элементы языка в макрос можно только путем ручной правки реестра или путем применения специальных программ/плагинов.

В макрокомандах $If и $While в качестве условия Expr могут использоваться операции сравнения:
Operator Description
== Равно
!= Не равно
< Меньше
> Больше
<= Меньше или равно
>= Больше или равно

ATTENTION!

  • Операции сравнения для строковых переменных регистрозависимые.

Булевое выражение Expr может быть сложным. В этом случае простые выражения объединяются с помощью следующих операторов:
Operator Description
&& Logical AND
|| Logical OR

Simple expressions can be enclosed in braces.
